# Welcome to My Vivino
***
## Task
Building a modern, data-driven wine recommendation system to replace the outdated rules-based engine.
## Description
By cleaning the wine dataset, generating analytical visualizations, computing value metrics, and building a hybrid ML recommendation model using numerical features and TF-IDF text similarity.
## Installation
Make sure to have python and jupyter with pandas and numpy and matplotlib and scikit-learn exetensions installed.
## Usage
```
python my_vivino.py
```
```
jupyter lab
```
